PROGETTAZIONE E SVILUPPO DI UN MODULO JAVA
Lo  scopo  del  progetto  è  lo  sviluppo  di  un componente  software  di  supporto  alla  gestione  di  un sistema di micro blogging. Astrattamente un micro blog è una collezione di contenuti testuali con una collezione di contenuti testuali con una lunghezza massima. Per semplicità assumiamo che i contenuti siano rappresentati da stringhe lunghe al massimo 140 caratteri.
Assumiamo, inoltre, che il micro blog sia usabile solo tramite una registrazione: solo gli utenti registrati possono scrivere i messaggi presenti nel micro blog, mentre la lettura è aperta.
Il  micro blog ha un'unica password, comune a tutti gli utenti. Il codice restituito dal metoro tweet è invece unico, nel senso che identifica in maniera univoca il messaggio. Tutti gli utenti che conoscono il codice possono rimuovere il messaggio.
Si  definisca la  classe User (con un’unica variabile d’istanza String nick) e le classi di eccezioni usate nell’interfaccia in modo da compilare l'interfaccia con successo. Il tipo generico List<E> è quello fornito dalla libreria standard di Java.
Si definisca un'implementazione dell'interfaccia SimpleTw, chiamata myTw. Il costruttore di myTw ha un argomento di tipo String che serve per fornire una password quando si crea un oggetto della classe. Non sono posti vincoli sulla struttura di implementazione prescelta.
